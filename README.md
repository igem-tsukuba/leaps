## 💻 Installation

```bash
# 1. リポジトリのクローン
$ git clone https://github.com/yushin-ito/leaps.git
$ cd leaps

# 2. 仮想環境の作成
$ python -m venv .venv
$ source .venv/bin/activate

# 3. 依存関係のインストール
$ pip install --upgrade pip
$ pip install -r requirements.txt
$ git clone https://huggingface.co/westlake-repl/SaProt_650M_AF2
```
